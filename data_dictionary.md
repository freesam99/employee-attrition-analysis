# Variables

This document describes each variable included in the IBM HR Analytics Employee Attrition & Performance dataset used in this project.

| Variable                     | Data Type | Description                                                     | Possible Values                                                                                                                                                                  |
| ---------------------------- | --------- | --------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Age**                      | Integer   | Employee's age in years.                                        | Integer (18–60)                                                                                                                                                                  |
| **Attrition**                | Text      | Indicates whether the employee left the company.                | Yes, No                                                                                                                                                                          |
| **BusinessTravel**           | Text      | Frequency of business travel.                                   | Non-Travel, Travel_Rarely, Travel_Frequently                                                                                                                                     |
| **DailyRate**                | Integer   | Employee's daily pay rate.                                      | Integer                                                                                                                                                                          |
| **Department**               | Text      | Department where the employee works.                            | Human Resources, Research & Development, Sales                                                                                                                                   |
| **DistanceFromHome**         | Integer   | Distance from home to work (miles).                             | Integer                                                                                                                                                                          |
| **Education**                | Integer   | Highest education level attained.                               | 1 = Below College, 2 = College, 3 = Bachelor, 4 = Master, 5 = Doctor                                                                                                             |
| **EducationField**           | Text      | Employee's field of study.                                      | Human Resources, Life Sciences, Marketing, Medical, Other, Technical Degree                                                                                                      |
| **EmployeeCount**            | Integer   | Number of employees represented by the record.                  | Always 1                                                                                                                                                                         |
| **EmployeeNumber**           | Integer   | Unique employee identifier.                                     | Integer                                                                                                                                                                          |
| **EnvironmentSatisfaction**  | Integer   | Satisfaction with the work environment.                         | 1 = Low, 2 = Medium, 3 = High, 4 = Very High                                                                                                                                     |
| **Gender**                   | Text      | Employee gender.                                                | Male, Female                                                                                                                                                                     |
| **HourlyRate**               | Integer   | Employee's hourly pay rate.                                     | Integer                                                                                                                                                                          |
| **JobInvolvement**           | Integer   | Level of involvement in their job.                              | 1 = Low, 2 = Medium, 3 = High, 4 = Very High                                                                                                                                     |
| **JobLevel**                 | Integer   | Organizational level of the employee.                           | 1–5                                                                                                                                                                              |
| **JobRole**                  | Text      | Employee's job title.                                           | Healthcare Representative, Human Resources, Laboratory Technician, Manager, Manufacturing Director, Research Director, Research Scientist, Sales Executive, Sales Representative |
| **JobSatisfaction**          | Integer   | Employee's satisfaction with their job.                         | 1 = Low, 2 = Medium, 3 = High, 4 = Very High                                                                                                                                     |
| **MaritalStatus**            | Text      | Marital status of the employee.                                 | Single, Married, Divorced                                                                                                                                                        |
| **MonthlyIncome**            | Integer   | Employee's monthly salary.                                      | Integer                                                                                                                                                                          |
| **MonthlyRate**              | Integer   | Monthly pay rate.                                               | Integer                                                                                                                                                                          |
| **NumCompaniesWorked**       | Integer   | Number of companies previously worked for.                      | Integer                                                                                                                                                                          |
| **Over18**                   | Text      | Indicates whether the employee is over 18 years old.            | Y                                                                                                                                                                                |
| **OverTime**                 | Text      | Indicates whether the employee works overtime.                  | Yes, No                                                                                                                                                                          |
| **PercentSalaryHike**        | Integer   | Percentage increase in salary from the previous year.           | Integer                                                                                                                                                                          |
| **PerformanceRating**        | Integer   | Employee's performance evaluation.                              | 3 = Excellent, 4 = Outstanding                                                                                                                                                   |
| **RelationshipSatisfaction** | Integer   | Satisfaction with workplace relationships.                      | 1 = Low, 2 = Medium, 3 = High, 4 = Very High                                                                                                                                     |
| **StandardHours**            | Integer   | Standard number of work hours represented in the dataset.       | Always 80                                                                                                                                                                        |
| **StockOptionLevel**         | Integer   | Employee's stock option level.                                  | 0–3                                                                                                                                                                              |
| **TotalWorkingYears**        | Integer   | Total years of professional work experience.                    | Integer                                                                                                                                                                          |
| **TrainingTimesLastYear**    | Integer   | Number of training sessions completed during the previous year. | Integer                                                                                                                                                                          |
| **WorkLifeBalance**          | Integer   | Employee's work-life balance rating.                            | 1 = Bad, 2 = Good, 3 = Better, 4 = Best                                                                                                                                          |
| **YearsAtCompany**           | Integer   | Number of years employed at the company.                        | Integer                                                                                                                                                                          |
| **YearsInCurrentRole**       | Integer   | Number of years in the employee's current role.                 | Integer                                                                                                                                                                          |
| **YearsSinceLastPromotion**  | Integer   | Number of years since the employee's last promotion.            | Integer                                                                                                                                                                          |
| **YearsWithCurrManager**     | Integer   | Number of years working with the current manager.               | Integer                                                                                                                                                                          |

---

## Notes

* The target variable for this project is **Attrition**, which indicates whether an employee has left the company.
* Variables such as **EmployeeCount**, **Over18**, and **StandardHours** contain constant values across all records and provide little analytical value for predictive analysis.
* Satisfaction-related variables use ordinal scales, where larger values indicate greater satisfaction.
* All monetary variables are stored as integers and represent employee compensation metrics.
